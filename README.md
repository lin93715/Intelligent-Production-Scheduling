# 基於基因演算法之智慧製造排程 API

## 📌 專案概述
** 智慧排貨 **
** 是一套以 **ASP.NET Core** 為基礎開發的 Web API，  
用於展示如何將 **基因演算法（Genetic Algorithm, GA）** 封裝為可供智慧製造系統呼叫的最佳化服務。

本專案主要用於：

- **智慧創新大賞相關競賽之系統展示**
- 排程與最佳化演算法之工程化實作範例
- 學術研究與產學應用之架構參考

⚠️ **基於公司商業機密與資訊安全政策，本 GitHub 公開版本已刻意移除以下內容：**

- 核心目標函數與關鍵最佳化公式  
- 公司內部使用之 SDK、套件與系統連線  
- 實際生產環境之商業規則與限制條件  

因此，本專案 **無法直接編譯並執行**，  
但 **系統架構、API 設計方式與演算法整合流程皆完整保留**。

## 🧠 設計理念

在實際的智慧製造環境中，  
最佳化演算法通常不會以單機程式或 Script 形式存在，而是：

- 以 **API 服務** 方式提供
- 與其他周邊系統整合
- 支援即時或準即時的排程請求

本專案的核心目標即在於示範：

> **如何將基因演算法設計為「企業級、可擴充、可維運」的最佳化服務**

強調以下工程與研究觀點：

- 演算法與系統邏輯分離（Separation of Concerns）
- 最佳化模組之可替換與可擴充設計
- 適用於智慧製造場域的服務化架構

---

## 🧬 基因演算法流程（概念說明）

雖然實際公式與商業邏輯已移除，本系統設計之 GA 流程符合標準最佳化框架：

1. **問題編碼（Encoding）**  
   - 將排程決策轉換為染色體表示法
2. **初始族群產生（Initialization）**  
   - 隨機或規則導向初始化
3. **適應值評估（Fitness Evaluation）**  
   - 單目標或多目標設計（已移除）
4. **選擇機制（Selection）**  
   - 輪盤法或競賽選擇法
5. **交配與突變（Crossover & Mutation）**  
   - 支援限制條件導向之操作
6. **終止條件（Termination）**  
   - 迭代次數或收斂條件
